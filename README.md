## 🧾 소개

바쁜 일상을 살아가다보면 어쩔 수 없이 성공과 효율에만 집중하게 됩니다.</br>

그런 강박 때문에 작은 도전도 하기 힘들때가 있습니다.</br>

도전을 하기 위해 큰 용기나 결단이 필요하고, 실패가 두렵기 때문에 하지 못한 일들이 많으셨죠?</br>

***오늘의 실패***로 하루에 했던 실패를 기록하고 실패로 얻은 것들을 회고하고</br>

영수증을 출력해 친구들과 공유해보세요.
